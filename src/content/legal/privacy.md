## 4. Pages CMS

**Pages CMS** is used to manage the content of this website. It is a content management system for static websites that is connected to GitHub and enables access to repository contents.

As part of the use of Pages CMS, personal data may be processed, in particular login data in connection with GitHub, user identifiers, email addresses in the context of collaborator access, as well as technical metadata required for authentication, access control, and content management.

Authentication is carried out via GitHub or via access tokens connected to GitHub or a GitHub App installation. The specific processing depends on the way access to the respective repository is carried out.

As a rule, Pages CMS does not affect the normal visitors of this website, but only persons who manage or edit content.

Further information:

- [Pages CMS on GitHub Marketplace](https://github.com/marketplace/pages-cms)
